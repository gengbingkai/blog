星期一, 四月 6日 2020, 12:00 凌晨

423 字 4 分钟

  1. 第一步，首先你要检测你是否安装node.js 

    node -v     #如果打印出版本号就说明已经安装

如果没安装node.js ,就访问 https://nodejs.org/en/ 并下载，安装

  1. 检验node.js的npm命令

    npm -v

  1. 安装Hexo

    npm install -g hexo-cli

    等待安装完成

    hexo -v  #检验是否安装成功

  2. 在 本地新建一个文件夹，用来存放个人博客，

    mkdir blog
cd blog   #进入到文件夹

  3. 新建Hexo项目

    hexo init my_blog  # my_blog=项目名称

    这个过程会比较慢，需要稍微等一会！

    建议在看到进度条的时候可以停掉，

    因为实在下载主题，时间会比较长

  4. 执行  npm istall

    安装一些配置文件 ，要等一等，不要急哦！！

  5. 以上执行完以后，就可以运行啦！！！

    hexo server    #简写 hexo s

    出现端口号就可以在本地浏览器访问啦！！

​ 出现页面就成功啦！！

  1. 接下来上传gitee码云

    8.1 登录码云新建仓库，输入仓库名称，点击公开，下拉点击创建。

  2. 利用编辑器Visual Studio Code或其他的都可以，打开_config.yml

  1. 生成打包文件

    在跟_config.yml同级的目录下执行命令

    hexo g   #把项目打包

  1. 在生成的public文件目录下

    git init    #初始化
git remote add origin  +  自己的仓库地址  #建立关系

    然后素质三连提交代码

    git add -A   #提交当前目录下的所有文件
git commit -m 'blog' # 备注
git push origin master

  1. 提交上以后

    点击服务，点击gitee pages

下滑，点击启动

生成路由就可以访问啦！！！

你的简易的个人博客搭建好啦
