迭代器

迭代器的定义

  1. 迭代器是访问集合内元素的方式，迭代器对象从集合的第一个元素开始访问，直到所有的元素都被访问一遍后结束

  2. 迭代器仅是一容器对象，它有两个基本方法

next方法：返回容器的下一个元素

iter方法：返回迭代器自身

使用迭代器的斐波那契

class fei(object):
    def __init__(self,n):
        self.num1 = 0
        self.num2 = 1
        self.n = n
        self.indexes = 0
    def __iter__(self):
        return self

    def __next__(self):
        if self.n > self.indexes:
            num = self.num1
            self.num1,self.num2 = self.num2,self.num1+self.num2
            self.indexes+=1
            return num
        else:
            raise StopIteration
if __name__ == '__main__':
    f = fei(10)
    print(list(f))

生成器

生成器的定义

生成器可以理解为一种数据类型，这种数据类型自动实现了迭代器协议（其他数据类型需要调用自己的内置iter方法）

在Python中，一边循环，一边计算的机制，称为生成器。

跟普通函数不同的是，生成器是一个返回迭代器的函数

在调用生成器运行的过程中，每次遇到 yield 时函数会暂停并保存当前所有的运行信息，返回 yield 的值, 并在下一次执行 next() 方法时从当前位置继续运行。

使用生成器简单实现斐波那契

def fei(n):
    num1,num2 = 0,1
    indexes = 0
    while n > indexes:
        num = num1
        num1,num2 = num2,num1+num2
        indexes += 1
        yield num

print(list(fei(10)))
print(fei(10))
