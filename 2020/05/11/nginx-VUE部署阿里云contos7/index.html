首先，说明我的版本

windows 10系统

阿里云的contos7

下载nginx

先版本一下命令

yum install nginx   #下载nginx到服务器

yum remove nginx #删除nginx

nginx
启动
systemctl start nginx
停止
systemctl stop nginx
重启
systemctl restart nginx

我的连接远程服务器的应用是

Xshell 和 Xftp6

Xshell用来连接

Xftp6 用来传输文件

把命令看完就是 VUE和nginx啦

如果页面不显示，说明路径有问题

打开在VUE项目的config/index.js文件里

  build: {
    // Template for index.html
    index: path.resolve(__dirname, '../dist/index.html'),

    // Paths
    assetsRoot: path.resolve(__dirname, '../dist'),
    assetsSubDirectory: 'static',
    assetsPublicPath: '/',

找到这些代码，把assetsPublicPath: ‘/‘,这一行改为

assetsPublicPath: './',

没错，就加了个点

然后，再找到build/utils.js文件

    if (options.extract) {
      return ExtractTextPlugin.extract({
        use: loaders,
        publicPath:'../../',
        fallback: 'vue-style-loader'
      })
    } else {
      return ['vue-style-loader'].concat(loaders)
    }

找到这些，并修改，自己找找哪里不同？然后修改

修改完VUE后就在项目文件下打包,执行下条命令

npm run build

完成后就会生成dist文件

用Xftp 传输到服务器相应位置

然后修改nginx

找到/etc/nginx/nginx.conf文件,你会看到这些
