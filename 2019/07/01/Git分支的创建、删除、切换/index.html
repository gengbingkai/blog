1、本地创建新分支并切换到新分支
# git checkout -b 分支名
2、提交分支到远程仓库
# git push origin 分支名
3、切换分支
# git checkout 分支名
4、删除远程分支
# git push origin --delete 分支名
5、删除本地分支
# git branch -D 分支名

Tips：其他分支命令

  • 查看当前分支 git branch
  • 查看所属分支 git branch -a
